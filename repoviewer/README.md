## GitHub Repositories Viewer

### Installation

1. **Clone the repository**
    ```bash
    git clone https://github.com/theoseidel/nexusflow-interview.git
    cd nexusflow-interview
    cd repoviewer
    ```

2. **Install dependencies**
    ```bash
    npm install
    ```

### Starting the App

1. **Start the development server**
    ```bash
    npm start
    ```

2. **Open**
    ```
    http://localhost:3000
    ```
